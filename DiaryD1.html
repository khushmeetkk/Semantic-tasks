DAY 1

Introduction to HTML and CSS

Today we started learning about the basics of HTML and CSS. HTML (HyperText Markup Language) is used to structure content on the web, while CSS (Cascading Style Sheets) is used for styling and layout.

Basics of HTML

We covered foundational HTML elements such as paragraphs, headings, lists, tables, and forms. These elements are essential for structuring and organizing content on web pages.

Semantic Web

The semantic web refers to adding meaning and context to web content using semantic HTML tags like <header>, <footer>, and <article>. This improves accessibility, SEO, and machine readability of web pages.

Using semantic HTML helps search engines understand the structure of your content better, which can lead to improved search engine rankings and a better user experience.

GitHub and GitHub Pages

GitHub is a platform for version control and collaboration. It allows developers to manage and track changes to their code, facilitating teamwork and project management.

GitHub Pages is a feature of GitHub that allows you to publish static websites directly from a GitHub repository. By simply pushing your HTML, CSS, and JavaScript files to a specific branch (usually the gh-pages branch), your website becomes accessible via a URL formatted like https://username.github.io/repository.

Today, we also tried to work on GitHub, creating web pages and pushing our HTML and CSS files to a GitHub repository to see our work live on the web.
