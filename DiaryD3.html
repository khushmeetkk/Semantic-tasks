DAY 3

Navigation Bar

Today, we learned about creating a navigation bar to improve the user experience by providing easy access to different sections of a website. The navigation bar typically includes links to other pages or sections within the same page.

A simple example of a navigation bar using an unordered list:

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#services">Services</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>

Anchor Tag

We explored the use of the anchor tag (<a>) to create hyperlinks. Anchor tags are essential for linking to other web pages or different sections within the same page.

An example of an anchor tag:

<a href="https://www.example.com">Visit Example</a>

Header and Footer

Fluid Concept Model

The fluid concept model refers to designing responsive web pages that adapt to different screen sizes and devices. This approach ensures that the layout and content adjust fluidly to provide a consistent user experience across various devices.

We used CSS techniques like flexible grids and media queries to implement fluid designs.
