DAY 2

Introduction to CSS

Today we delved into Cascading Style Sheets (CSS), which is used for styling and formatting web pages. CSS allows us to control the appearance of HTML elements.

Types of CSS

We discussed the different types of CSS:

  • Inline CSS: Applied directly to an HTML element using the style attribute. For example:
    • <p style="color: blue; font-size: 16px;">This is a paragraph with inline CSS.</p>
  • Internal CSS: Placed within the <style> tag in the <head> section of an HTML file. For example:
    • <style>
      p {
        color: blue;
        font-size: 16px;
      }
    </style>
  • External CSS: Stored in separate CSS files and linked to HTML documents using the <link> tag. For example:
    • <link rel="stylesheet" type="text/css" href="styles.css">

Best Practices of External CSS

We discussed best practices for using External CSS:

  • Keep styles separate from content for better maintainability and scalability.
  • Use meaningful class and ID names to make your CSS easier to understand and modify.
  • Group related styles together to improve organization.

Box Model and <div> Tags

The CSS Box Model describes the space around and within HTML elements. It consists of margins, borders, padding, and the actual content area.

<div> tags are used as containers to group and style HTML elements, allowing us to structure and organize content on web pages effectively.

Practical Tasks

During today's session, we practiced using containers, headers, and other <div> tags to create structured layouts and apply CSS styles to enhance the visual appearance of our web pages.
